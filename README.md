# minimemo
minimemo 는 SNS 기능이 빠진 개인용 한줄 블로그다. 잠깐동안 내 마음 안에 머물렀다 금새 떠나버리는 생각이나 아이디어를 텍스트 형식으로 저장할 수 있다


### 개발환경
* Back-end : [firebase](https://firebase.google.com)
* Front-end : es6, requirejs, babel


### 기능설명
* 단축키
  * Alt(or meta) + W : 글쓰기
  * Alt(or meta) + S : 검색
  * Alt(or meta) + Enter : 저장/조회


### 이후 추가될 기능
* 내용 저장시 애니메이션 효과
